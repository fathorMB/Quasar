# Quasar

Quasar is a modular CQRS + event-sourcing micro-framework. This package provides one of the framework components. See the [project repository](https://github.com/fathorMB/Quasar) for full documentation, samples, and release notes.

## Getting Started

```powershell
# add the package from GitHub Packages
Install-Package Quasar.Core -Source "https://nuget.pkg.github.com/fathorMB/index.json"
```

Refer to the main README for usage examples and architectural guidance.
